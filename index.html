Parameters
Supply voltage in volts. This is the logic-high rail driven onto the VCC port and the maximum voltage used in the transient plots.
Threshold voltage of enhancement nMOS transistors. Larger values make pull-down devices turn on later and weaker for the same gate voltage.
Threshold voltage of depletion-load devices. Because it is normally negative, the load behaves partly on even with gate bias near zero.
Sheet resistance of p diffusion in ohms per square. Higher values make p diffusion runs slower and more resistive.
Sheet resistance of n diffusion in ohms per square. This sets the resistance of n diffusion paths outside the active transistor channel model.
Sheet resistance of polysilicon in ohms per square. This affects gate routing delay and any poly signal spine used in the layout.
Capacitance added per p-diffusion cell in femtofarads. It models distributed parasitic loading from each p region cell.
Capacitance added per n-diffusion cell in femtofarads. Larger values slow node transitions on n-diffusion regions.
Capacitance added per poly cell in femtofarads. This mostly affects gate and routing delay for long poly wires.
Capacitance added per metal1 cell in femtofarads. It represents wiring parasitics of the first metal layer.
Capacitance added for each p-to-n parasitic junction edge in femtofarads. This is the depletion capacitance at diffusion boundaries.
Capacitance added by each contact between metal and diffusion or poly in femtofarads.
Transistor strength coefficient per extracted unit width. Larger values make conducting transistors stronger and faster.
Channel-length modulation factor. It changes how much transistor current still depends on drain voltage after turn-on.
Strength scaling used for depletion-mode loads relative to the nominal extracted device strength.
